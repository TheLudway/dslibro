Capítulo 37 Accediendo al terminal e instalando Git

Antes de comenzar, debemos asegurarnos de que tengan acceso a un terminal y que Git esté instalado. El terminal está integrado en los sistemas Mac y Linux, pero los usuarios de Windows deberán instalar un emulador. Hay muchas opciones de emulador disponibles, pero aquí mostramos cómo instalar Git Bash porque se puede hacer como parte de la instalación de Windows Git. Debido a las diferencias entre Mac y Windows, las secciones de este capítulo se dividen tomando esto en cuenta.

37.1 Accediendo al terminal en una Mac

En el Capitulo 38, describimos cómo el terminal es nuestra ventana al mundo de Unix. En una Mac, pueden acceder a un terminal abriendo la aplicación en la carpeta Utilities:

También pueden usar la aplicación Spotlight en la Mac presionando comando-barra espaciadora y entonces escribiendo Terminal.

Otra forma de acceder al terminal es desde RStudio. En el panel Console, deberían ver una pestaña Terminal. Si hacen clic en esa pestaña, abrirá una ventana de terminal.

37.3 Instalación de Git y Git Bash en Windows

Advertencia: Las instrucciones en esta subsección no son para usuarios de Mac.

Hay varios programas de software que les permitirán ejecutar comandos de Unix en Windows. Usaremos Git Bash ya que interactúa con RStudio y se instala automáticamente cuando instalamos Git for Windows.

  1. Comiencen buscando Git for Windows en su navegador y haciendo clic en el enlace de git-scm.com.

  1. Esto los llevará a la página Download Git donde pueden descargar el maintained build mas reciente:

  1. Entonces pueden aceptar ejecutar el instalador y aceptar la licencia:

  1. En uno de los pasos de la instalación, se les pedirá que elijan el editor predeterminado para Git. A menos que ya sean usuarios de vi o vim, no les recomendamos que seleccionen vim, que puede ser el predeterminado. Si no reconocen un editor con el que están familiarizados entre las opciones, les recomendamos que seleccionen nano como su editor predeterminado para Git, ya que es el más fácil de aprender:

  1. La siguiente decisión de instalación es una muy importante. Este proceso de instalación instala Git Bash. Recomendamos que seleccionen Git and optional Unix tools from the Windows Command ya que esto les permitirá aprender Unix desde RStudio. Sin embargo, si hacen esto, algunos comandos que se ejecutan en su línea de comandos de Windows dejarán de funcionar. Si no usan su línea de comandos de Windows, entonces esto no será un problema. Además, la mayoría, si no todas, de estas líneas de comando de Windows tienen un equivalente de Unix que podrán usar.

  1. Ahora pueden continuar seleccionando las opciones predeterminadas.

Ya han instalado Git en Windows.
