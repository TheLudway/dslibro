Prefacio

Este libro comenzó como las notas utilizadas para enseñar la clase de HarvardX Data Science Series1.

El código Rmarkdown que se usó para generar el libro está disponible en GitHub2. Tengan en cuenta que el tema gráfico utilizado para los gráficos a lo largo del libro se pueden recrear utilizando la función ds_theme_set() del paquete dslabs.

Un PDF de la versión en inglés de este libro está disponible en Leanpub3.

Una copia impresa de la versión en inglés de este libro está disponible en CRC Press4.
