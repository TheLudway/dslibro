36.1 Instalando R

RStudio es un entorno de escritorio interactivo, pero no es R, ni incluye R cuando lo descarga e instala. Por lo tanto, para usar RStudio, primero necesitamos instalar R.

  1. Puede descargar R desde la Red Integral de Archivo R (CRAN)119. Busque CRAN en su navegador:

  1. Una vez en la página CRAN, seleccione la versión para su sistema operativo: Linux, Mac OS X o Windows.

Aquí mostramos capturas de pantalla para Windows, pero el proceso es similar para las otras plataformas. Cuando difieren, también mostraremos capturas de pantalla para Mac OS X.

  1. Una vez en la página de descarga de CRAN, tendrá varias opciones. Desea instalar el subdirectorio base. Esto instala los paquetes básicos que necesita para comenzar. Más adelante aprenderemos cómo instalar otros paquetes necesarios desde R, en lugar de hacerlo desde esta página web.

  1. Haga clic en el enlace de la última versión para iniciar la descarga.

  1. Si está utilizando Chrome, en la parte inferior de su navegador debería ver una pestaña que le muestra el progreso de la descarga. Una vez que se descarga el archivo instalador, puede hacer clic en esa pestaña para comenzar el proceso de instalación. Otros navegadores pueden ser diferentes, por lo que tendrá que encontrar dónde almacenan los archivos descargados y hacer clic en ellos para comenzar el proceso.

Si usa Safari en una Mac, puede acceder a la descarga a través del botón de descarga.

  1. Ahora puede hacer clic en diferentes opciones para finalizar la instalación. Le recomendamos que seleccione todas las opciones predeterminadas.

Seleccione el valor predeterminado incluso cuando reciba una advertencia ominosa.

Al seleccionar el idioma, tenga en cuenta que será más fácil seguir este libro si selecciona inglés.
